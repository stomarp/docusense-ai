# DocuSense AI MVP Scope

## MVP Goal

Build a working document risk analyzer where a user can upload a document and receive a structured risk report.

## Phase 1 MVP Features

- FastAPI backend
- PostgreSQL database
- SQLAlchemy models
- Docker database setup
- Document upload endpoint
- Text extraction
- Rule-based risk analyzer
- ML section classifier
- Risk score
- Missing sections
- Risky phrases
- Report history

## Phase 2 MVP Features

- Document type detection
- Multi-document support:
  - HR policy
  - Lease agreement
  - Offer letter
  - Contract
  - Compliance document
- Key obligations
- Red flags
- Suggested improvements
- Plain-English summary

## Not Included Yet

- Authentication
- Payments
- Team accounts
- PDF export
- Production deployment
- Advanced LLM analysis

These will be added after the core backend and frontend flow is stable.
