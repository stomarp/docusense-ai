# DocuSense AI Product Plan

## Product Name

DocuSense AI

## Previous Name

DocuGuard HR

## Product Goal

Build an end-to-end AI document risk analyzer that helps users understand risk in HR policies, leases, contracts, offer letters, vendor agreements, and compliance documents.

## Rebrand Reason

DocuGuard HR was focused mainly on HR policy compliance. DocuSense AI expands the product into a broader document intelligence platform that can analyze many types of documents.

## Target Users

- Job seekers reviewing offer letters
- Tenants reviewing lease agreements
- Small businesses reviewing HR policies
- Founders reviewing vendor contracts
- Compliance teams reviewing internal documents
- Non-technical users who need plain-English document review

## Core User Flow

Upload Document
-> Extract Text
-> Detect Document Type
-> Analyze Risk
-> Generate Plain-English Report
-> Export Report

## Core Features

- Document upload
- Text extraction
- Document type detection
- Plain-English summary
- Risk score
- Missing sections
- Key obligations
- Red flags
- Suggested improvements
- Exportable report

## MVP Strategy

Start with the existing DocuGuard HR backend and broaden it step by step.

Phase 1 keeps the existing rule-based and ML foundation.

Phase 2 adds multi-document risk analysis.

Phase 3 adds frontend.

Phase 4 upgrades analysis with LLM reasoning.
