# DocuSense AI Frontend

Next.js frontend for DocuSense AI.

## Local Development

Backend should run on port 8010.

Frontend runs on port 3001.

Commands:

npm install
npm run dev

Open:

http://localhost:3001

## API

By default, the frontend uses:

http://127.0.0.1:8010

To override:

NEXT_PUBLIC_API_BASE_URL=http://127.0.0.1:8010
